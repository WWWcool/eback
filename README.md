# Boilerplate for game backend

> This repository is accompanied by a https://github.com/wmnnd/nginx-certbot.git.

## Installation
1. [Install docker-compose](https://docs.docker.com/compose/install/#install-compose).

2. Run server:
`docker-compose up`
